# irocn-os
irocn-os, 是基于Ubuntu 19.04版本针对Filecoin/Storj优化过的操作系统。操作系统轻量级、安全策略优化、磁盘I/O优化、磁盘阵列

## irocn-os 功能
- 自动安装
- 内置Filecoin/Storj miner客户端
- 自动更新软件包
- 内置Filecon、Storj客户端运行状况数据采集
- 磁盘自动检测和mount
- 磁盘阵列优化
- 支持远程XPE安装
- 定制启动界面
